# huayra-tdapplet

Ícono de notificación que reporta el estado del TheftDeterrent 

Para bajarlo y probarlo ejecutá `git clone https://github.com/HuayraLinux/huayra-tdapplet && cd huayra-tdapplet && gjs tdapplet.js`
